# Shared

This directory contains shared resources, types, and utilities used across the frontend and backend.

## Structure

```
shared/
├── types/        → Shared TypeScript and Python type definitions
└── constants/    → Shared constants (e.g. error codes, status values)
```

## Usage

Shared types ensure consistency between the frontend and backend API contracts. Any interface or enum used by both services should be defined here.
